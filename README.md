# 학폭케어 (HakPoKe)

학교폭력 예방 및 신고 시스템 웹사이트

## 프로젝트 소개

학폭케어는 학교폭력을 예방하고 피해자를 보호하기 위한 정보 제공 및 익명 신고 시스템입니다.

**"혼자가 아닙니다"** - 학교폭력으로 고통받는 학생들에게 희망과 실질적인 도움을 제공합니다.

### 주요 기능

- **감성적 스토리텔링**: 피해자의 관점에서 공감하고 희망을 전달하는 콘텐츠
- **희망의 이야기** (`/hope`): 학교폭력을 극복한 실제 이야기를 통한 용기와 희망 제공
- **단계별 도움 가이드** (`/help`): 5단계로 구성된 구체적이고 실천 가능한 도움받기 방법
- **정보 제공** (`/info`): 학교폭력의 유형, 대처 방법, 도움받을 수 있는 기관 정보
- **신고 시스템** (`/report`): 익명 또는 실명으로 학교폭력을 신고할 수 있는 양식
- **긴급 연락처**: 117(학교폭력신고센터), 112(경찰) 등 긴급 연락처 안내

### 디자인 철학

이 프로젝트는 단순한 정보 제공을 넘어, **피해자에게 진심으로 공감하고 희망을 전달**하는 것을 목표로 합니다:

- 어둠에서 빛으로 전환되는 그라데이션 (고통 → 희망의 시각적 표현)
- 강렬하고 감성적인 메시지와 타이포그래피
- 피해자의 감정에 공감하는 따뜻한 톤앤매너
- 즉각적인 행동을 유도하는 명확한 CTA (Call-to-Action)

## 기술 스택

- **Framework**: Next.js 16 (React 19)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Linting**: ESLint

## 시작하기

### 필수 요구사항

- Node.js 18.17 이상

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

개발 서버가 실행되면 [http://localhost:3000](http://localhost:3000)에서 확인할 수 있습니다.

### 빌드

```bash
# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm run start
```

## 프로젝트 구조

```
hakpoke/
├── app/                    # Next.js App Router
│   ├── layout.tsx         # 루트 레이아웃
│   ├── page.tsx           # 홈 페이지
│   ├── globals.css        # 글로벌 스타일

│       └── page.tsx
├── components/            # 재사용 가능한 컴포넌트
├── public/                # 정적 파일
└── package.json
```

## 주요 페이지

- **홈 (`/`)**: 감성적인 히어로 섹션과 희망의 메시지로 시작하는 메인 페이지
- **희망의 이야기 (`/hope`)**: 학교폭력 피해자가 극복한 진짜 이야기를 통한 용기 전달
- **도움받는 방법 (`/help`)**: 5단계로 구성된 구체적이고 실천 가능한 가이드
  1. 즉시 안전 확보
  2. 증거 확보하기
  3. 신뢰할 수 있는 어른에게 알리기
  4. 공식 신고하기
  5. 심리적 치유받기
- **정보 (`/info`)**: 학교폭력의 정의, 유형, 대처 방법, 도움받을 수 있는 곳
- **신고하기 (`/report`)**: 익명/실명 신고 폼 (긴급 연락처 강조)

## 중요 안내

⚠️ 이 프로젝트는 데모/교육 목적으로 제작되었습니다. 실제 학교폭력 신고는 다음 기관을 이용해주세요:

- **학교폭력신고센터**: 117
- **경찰 신고**: 112
- **청소년사이버상담센터**: 1388
