# 학폭케어 (HakPoKe)

학교폭력 예방 및 신고 시스템 웹사이트

## 프로젝트 소개

학폭케어는 학교폭력을 예방하고 피해자를 보호하기 위한 정보 제공 및 익명 신고 시스템입니다.

### 주요 기능

- **정보 제공**: 학교폭력의 유형, 대처 방법, 도움받을 수 있는 기관 정보 제공
- **신고 시스템**: 익명 또는 실명으로 학교폭력을 신고할 수 있는 양식 제공
- **긴급 연락처**: 117(학교폭력신고센터), 112(경찰) 등 긴급 연락처 안내

## 기술 스택

- **Framework**: Next.js 16 (React 19)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Linting**: ESLint

## 시작하기

### 필수 요구사항

- Node.js 18.17 이상

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

개발 서버가 실행되면 [http://localhost:3000](http://localhost:3000)에서 확인할 수 있습니다.

### 빌드

```bash
# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm run start
```

## 프로젝트 구조

```
hakpoke/
├── app/                    # Next.js App Router
│   ├── layout.tsx         # 루트 레이아웃
│   ├── page.tsx           # 홈 페이지
│   ├── globals.css        # 글로벌 스타일
│   ├── info/              # 정보 제공 페이지
│   │   └── page.tsx
│   └── report/            # 신고 시스템 페이지
│       └── page.tsx
├── components/            # 재사용 가능한 컴포넌트
├── public/                # 정적 파일
└── package.json
```

## 주요 페이지

- **홈 (`/`)**: 학폭케어 소개 및 주요 기능 안내
- **정보 (`/info`)**: 학교폭력 정보, 유형, 대처 방법
- **신고하기 (`/report`)**: 익명/실명 신고 폼

## 중요 안내

⚠️ 이 프로젝트는 데모/교육 목적으로 제작되었습니다. 실제 학교폭력 신고는 다음 기관을 이용해주세요:

- **학교폭력신고센터**: 117
- **경찰 신고**: 112
- **청소년사이버상담센터**: 1388
